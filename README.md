# ⚡ Iot Base Smart Energy Meter (ESP32 IoT Project)

Refer to:`(Demo Image.pdf)`

## 📌 About the Project
An IoT-based Smart Energy Meter using ESP32 that monitors Voltage, Current, Power, Energy, and Cost in real-time.  
Data is displayed on an LCD and remotely monitored using the Blynk app.

---

## ✨ Features
- ⚡ Real-time Voltage & Current monitoring  
- 📊 Power & Energy calculation  
- 💰 Cost estimation  
- 📟 LCD Display output  
- 📱 Blynk App integration  
- ☁️ Cloud monitoring  

---

## 🛠️ Components Used
- ESP32  
- Voltage Sensor  
- Current Sensor  
- 16x2 I2C LCD  
- Relay Module  
- Breadboard & Wires

## ⚙️ Setup Instructions
1. Install Arduino IDE  
2. Add ESP32 Board  
3. Install Libraries (Blynk, LiquidCrystal_I2C)  
4. Upload Code to ESP32  
5. Connect components as per circuit
